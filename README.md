# نظام طلبات التوريد - مؤسسة المقام التجارية 2026

نظام متكامل لإدارة طلبات التوريد مع دعم متعدد الأصناف والمقاسات، وإنشاء تقارير PDF، والمشاركة عبر واتساب.

## ✨ الميزات

- **إدارة الأصناف**: إضافة/تعديل/حذف الأصناف مع اختيار ألوان للتمييز.
- **إدخال الكميات**: ستة مقاسات رئيسية (54-64) وستة مقاسات فرعية (S-3XL) لكل صنف.
- **حفظ تلقائي**: تخزين البيانات في `localStorage` لاستعادتها بعد إغلاق المتصفح.
- **تصدير PDF**: إنشاء تقارير مفصلة متعددة الصفحات مع ملخصات وإجماليات.
- **مشاركة واتساب**: إرسال الطلب للعميل أو المستودع بنقرة واحدة.
- **تصميم متجاوب**: يعمل على جميع الأجهزة (جوال، جهاز لوحي، حاسوب).

## 🛠️ التقنيات المستخدمة

- HTML5 / CSS3 / JavaScript (ES6)
- مكتبات خارجية:
  - [Font Awesome 6](https://fontawesome.com/) للأيقونات.
  - [jsPDF](https://github.com/parallax/jsPDF) لإنشاء ملفات PDF.
  - [html2canvas](https://html2canvas.hertzen.com/) لتحويل HTML إلى صورة.

## 🚀 كيفية الاستخدام

1. **تعبئة بيانات العميل** (الاسم، العنوان، المندوب) – حقول إلزامية.
2. **إدخال الكميات** في الجدول الرئيسي باستخدام أزرار `+/-` أو الكتابة المباشرة.
3. **إدارة الأصناف**: زر "إضافة/تعديل أصناف" لإضافة صنف جديد أو تعديل اسم/رمز/لون أي صنف.
4. **إنشاء PDF**: زر "تصدير PDF مفصل" – سيتم التحقق من الحقول قبل التصدير.
5. **مشاركة واتساب**: زر "إرسال الطلب" ثم اختيار الوجهة (عميل/مستودع).
6. **إعادة تعيين**: مسح جميع البيانات المدخلة.

## 📁 رفع المشروع على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub.
2. ارفع الملفات التالية إلى المستودع:
   - `index.html`
   - `README.md`
   - `.gitignore` (اختياري)
3. اذهب إلى **Settings > Pages**، واختر فرع `main` (أو `master`) والمجلد `/ (root)`.
4. اضغط **Save** – سيظهر رابط التطبيق خلال دقيقة.
